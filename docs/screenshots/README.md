# Скриншоты из Grafana

Эта директория содержит скриншоты из Grafana, демонстрирующие работу OpenTelemetry инструментации.

## Как сделать скриншоты

1. Запустите Docker Compose:
   ```bash
   docker-compose up -d
   ```

2. Запустите приложение:
   ```bash
   npm run dev
   ```

3. Выполните несколько действий в приложении:
   - Откройте главную страницу
   - Выполните поиск товаров
   - Добавьте товары в корзину
   - Удалите товары из корзины

4. Откройте Grafana: http://localhost:3001

5. Перейдите в раздел **Explore** (иконка компаса слева)

6. Выберите источник данных **Tempo**

7. Сделайте скриншоты:
   - **trace-timeline.png** - Временная шкала трейсов
   - **service-graph.png** - Граф зависимостей сервисов
   - **metrics.png** - Метрики производительности

## Необходимые скриншоты

- [ ] `trace-timeline.png` - Timeline с трейсами операций
- [ ] `service-graph.png` - Service Graph
- [ ] `metrics.png` - Дашборд с метриками

Сохраните скриншоты в эту директорию.
